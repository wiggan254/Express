
Express - Endless Driving Game

Structure:
- index.html: Game launch screen
- garage.html: Vehicle selection and customization
- game.js: Main game logic
- style.css: UI styling
- assets/: Folder with placeholders for cars, music, voice, etc.
- admin/: Admin panel logic and HTML

To deploy:
1. Push to GitHub
2. Connect to Vercel or preferred static host
3. Update assets with real content (cars, music, voice, etc.)
